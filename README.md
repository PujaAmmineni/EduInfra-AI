# EduInfra-AI  
**AI Cyber-Infrastructure for Construction Education**

EduInfra-AI is an AI-powered web platform developed as part of a research and workforce-development initiative to support construction education using cognitive and generative AI. The system provides tools for ingesting learning materials, processing documents, and delivering AI-assisted educational workflows through a web interface.

This repository contains the core backend services, web UI, and AI pipelines used to support the platform.
